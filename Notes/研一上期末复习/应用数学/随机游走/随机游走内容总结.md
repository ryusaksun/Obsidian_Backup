根据提供的复习大纲和北大的随机过程课件（特别是北大随机过程课件：第-2-章-第-1-讲_随机游动 例子作业.pdf），随机游走（Random Walk）是随机过程部分的主题 1，也是考试的重点。复习大纲明确要求掌握前面四个问题以及期望加方差加概率分布特征。

以下是对随机游走核心内容的详细总结：

## 随机游走模型定义

设有质点在 $x$ 轴上作随机游动，初始时刻 $t=0$ 时位于原点。在 $t=1, 2, 3, \dots$ 时刻：

- 沿正方向移动一步（+1）的概率为 $p$。
- 沿反方向移动一步（-1）的概率为 $q = 1-p$。
- 记第 $n$ 步的状态（位置）为 $\eta_n$。这是一个离散时间、离散状态的随机过程。

## 第一个问题：经过 $n$ 步后的位置特征

这是复习大纲中特别强调的期望加方差加概率分布特征部分。

### 1. 概率分布

质点移动 $n$ 步后，若要到达位置 $m$（$m$ 与 $n$ 同奇偶，且 $|m| \le n$），假设向右移动了 $k$ 步，向左移动了 $n-k$ 步，则有 $k - (n-k) = m$，解得 $k = \frac{n+m}{2}$。

- 一维概率分布公式：

    $$P(\eta_n = m) = C_n^{\frac{n+m}{2}} p^{\frac{n+m}{2}} q^{\frac{n-m}{2}}$$

    这实际上是二项分布的一种变形。

### 2. 数字特征（重点记忆）

- 数学期望（均值）：

    $$E[\eta_n] = n(p-q)$$

    解释：如果 $p=q=1/2$（对称随机游走），期望为 0；否则会向概率大的一侧漂移。

- 方差：

    $$D[\eta_n] = 4npq$$

    解释：方差随步数 $n$ 线性增加，扩散范围越来越大。

- 相关函数（了解即可）：

    $$E[\eta_n \eta_m] = 4pq \min(n,m) + nm(p-q)^2$$

    如果是对称随机游走（$p=q=1/2$），则简化为 $\min(n,m)$。

## 第二个问题：经过 $n$ 步返回原点的概率（$u_n$）

质点要返回原点，必须向右和向左走的步数相等，因此总步数 $n$ 必须为偶数。

- 若 $n$ 为奇数：返回原点概率为 0。
- 若 $n=2m$（偶数）：

    $$u_{2m} = P(\eta_{2m}=0) = C_{2m}^m p^m q^m$$

    这是第 $2m$ 步处于原点的概率（不要求是第一次）。

## 第三个问题：经过 $n$ 步第一次返回原点的概率（$v_n$）

记 $v_{2n}$ 为第 $2n$ 步首次返回原点的概率。

- 与 $u_n$ 的关系（卷积公式）：

    $$u_{2n} = \sum_{k=1}^{n} v_{2k} u_{2n-2k}$$

    物理意义：第 $2n$ 步处于原点（$u_{2n}$）的事件，可以分解为在第 $2k$ 步（$k=1 \dots n$）第一次回到原点，然后在剩下的 $2n-2k$ 步里又回到了原点（可能是多次）。

- 矩生成函数（解题关键工具）：利用生成函数 $U(z) = \sum u_{2n}z^{2n}$ 和 $V(z) = \sum v_{2n}z^{2n}$，可以推导出重要关系：

    $$V(z) = 1 - \frac{1}{U(z)}$$

    由此解得第一次返回原点的生成函数为<font color="#ff0000">★</font>：

    $$V(z) = 1 - \sqrt{1 - 4pqz^2}$$

    利用泰勒展开可以求出 $v_{2n}$ 的通项公式。

## 第四个问题：迟早返回原点的概率与平均时间

这涉及系统的常返性（Recurrence）。

### 1. 迟早返回的概率 $P$

即粒子最终能回到原点的总概率。

$$P = \sum_{n=1}^{\infty} v_{2n} = V(1) = 1 - |p-q|$$

- 对称随机游走（$p=q=1/2$）：$P = 1$。说明必然会返回原点（常返态）。
- 非对称随机游走（$p \ne q$）：$P = 1 - |p-q| < 1$。说明有一定概率永远回不来（非常返态）。

### 2. 平均返回时间 $\mu$

即第一次返回原点所需步数的期望。

- 结论：即使在对称随机游走中（必然返回），平均返回时间也是无穷大（$\mu = \infty$）。这是随机游走的一个反直觉的重要性质。

## 总结与复习建议

根据复习大纲的要求：

1. 熟练掌握公式：$P(\eta_n=m)$ 的分布公式，以及 $E[\eta_n]$ 和 $D[\eta_n]$ 的计算公式。
2. 理解逻辑：理解处于原点（$u_n$）和首次返回原点（$v_n$）的区别，以及它们之间通过卷积公式联系的逻辑。
3. 常返性结论：记住当 $p=1/2$ 时是常返的（概率为 1），但平均时间无限；当 $p \ne 1/2$ 时是非常返的（概率<1）。
4. 计算练习：复习课件中关于利用生成函数求解 $u_n$ 和 $v_n$ 的例子。
