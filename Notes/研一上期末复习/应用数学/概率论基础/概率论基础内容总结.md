根据提供的 PPT 资料，这是《应用随机过程》课程中关于概率论基础的复习内容。以下是对该部分核心知识点的全面详细总结：

## 概率空间与基本概念

这部分构建了概率论的公理化体系。

1. 随机试验与样本空间

    - 随机试验：具有可重复性、可预见性（知道所有可能结果）但结果不确定性的试验。
    - 样本空间（$\Omega$）：随机试验所有可能基本结果的集合。$\Omega$ 中的元素称为样本点（$e$）。
    - 事件（$F$）：$\Omega$ 的子集，构成事件域。包括必然事件（$\Omega$）和不可能事件（$\emptyset$）。

2. 概率公理与性质

    - 概率定义：映射 $P: F \to R$，满足三条公理：
        1. 非负性：对任意事件 $A$，$0 \le P(A) \le 1$。
        2. 规范性：$P(\Omega) = 1$。
        3. 可列可加性：若 $A_1, A_2, \dots$ 互不相容，则 $P(\cup A_i) = \sum P(A_i)$。

    - 重要性质：$P(\emptyset)=0$；$P(B\backslash A) = P(B) - P(A)$（若 $A \subseteq B$）；以及概率的连续性（单调序列的极限概率等于极限集合的概率）。

3. 条件概率与独立性

    - 乘法公式：$P(AB) = P(B|A)P(A)$。
    - 全概率公式：若 $A_i$ 为完备事件族，则 $P(B) = \sum P(B|A_i)P(A_i)$。
    - 独立性：事件 $A, B$ 独立当且仅当 $P(AB) = P(A)P(B)$。对于多个事件，要求任意子集都满足乘积规则。

## 随机变量及其分布

这部分主要讨论如何用数学函数描述随机现象。

1. 定义与分布函数

    - 随机变量（RV）：定义在样本空间上的单值实函数 $X(e)$。
    - 分布函数（CDF）：$F_X(x) = P{X \le x}$。
    - CDF 性质：单调不减；右连续；$F(-\infty)=0, F(+\infty)=1$。

2. 随机变量的类型

    - 离散型：通过分布律 $P(X=x_k) = p_k$ 描述。
    - 连续型：通过概率密度函数（PDF）$f(x)$ 描述，满足 $F(x) = \int_{-\infty}^x f(t)dt$。
    - 混合型：上述两种的组合。

3. 典型分布（重点掌握）:

    - 离散型：
        - 0-1 分布（Bernoulli）。
        - 二项分布 $B(n,p)$：$P(X=k) = C_n^k p^k q^{n-k}$。
        - 泊松分布 $P(\lambda)$：$P(X=k) = \frac{\lambda^k}{k!}e^{-\lambda}$。
        - 几何分布 $G(p)$：具有无记忆性。
        - 超几何分布 $H(n, N, M)$。

    - 连续型：
        - 均匀分布 $U(a,b)$：密度为常数 $\frac{1}{b-a}$。
        - 指数分布 $E(\lambda)$：$f(x) = \lambda e^{-\lambda x}$ ($x>0$)，具有无记忆性（$P{X>s+t|X>s} = P{X>t}$）。
        - 正态分布 $N(\mu, \sigma^2)$：PDF 为 $\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$。
        - 瑞利分布（Rayleigh）与莱斯分布（Rician）：常出现在信号幅度分析中（如正态分布的模）。

## 多维随机变量

1. 联合分布

    - 联合 CDF：$F(x_1, \dots, x_n) = P{X_1 \le x_1, \dots, X_n \le x_n}$。
    - 性质：对每个变量单调不减、右连续；矩形不等式（如二维中 $F(b_1, b_2) - F(a_1, b_2) - F(b_1, a_2) + F(a_1, a_2) \ge 0$）。

2. 边缘分布与独立性

    - 边缘分布：通过对其他变量求和（离散）或积分（连续）得到。例如 $f_X(x) = \int_{-\infty}^{+\infty} f(x,y) dy$。
    - 独立性：$n$ 个随机变量相互独立 $\Leftrightarrow$ 联合分布等于边缘分布的乘积，即 $F(x_1, \dots, x_n) = \prod F_{X_i}(x_i)$ 或 PDF 满足 $f(x_1, \dots, x_n) = \prod f_{X_i}(x_i)$。

3. 随机变量的函数

    - 一元变换：若 $Y=g(X)$ 单调，可用反函数法求 PDF；线性变换 $Y=aX+b$ 的 PDF 也有特定公式。
    - 二元变换：$Z=g(X,Y)$，如卷积公式（和的分布）、最大值/最小值的分布。
    - 正态变量的线性变换：正态随机变量的线性组合仍然是正态分布。

## 数字特征

1. 数学期望（Expectation）

    - 定义：$E(X) = \sum x_k p_k$（离散）或 $\int xf(x)dx$（连续）。
    - 性质：线性性 $E(aX+bY) = aE(X) + bE(Y)$；若 $X,Y$ 独立，则 $E(XY) = E(X)E(Y)$。

2. 方差（Variance）

    - 定义：$D(X) = E[(X - EX)^2] = E(X^2) - (EX)^2$。
    - 性质：若 $X,Y$ 独立，$D(X+Y) = D(X) + D(Y)$。

3. 协方差与相关系数

    - 协方差：$Cov(X,Y) = E[(X-EX)(Y-EY)] = E(XY) - EXEY$。
    - 相关系数：$\rho_{XY} = \frac{Cov(X,Y)}{\sqrt{DX \cdot DY}}$。
    - 关系：
        - $\rho=0$ $\Leftrightarrow$ 不相关（Uncorrelated）。
        - $E(XY)=0$ $\Leftrightarrow$ 正交（Orthogonal）。
        - 独立（Independent）一定不相关；不相关不一定独立（除非是正态分布）。

4. 重要不等式

    - 切比雪夫不等式（Chebyshev）：描述随机变量偏离均值的概率界限。
    - 柯西-许瓦兹不等式（Cauchy-Schwarz）：$|E(XY)|^2 \le E(X^2)E(Y^2)$。

## 特征函数与母函数（变换域分析）

这部分是处理随机变量和、矩的强有力工具。

1. 特征函数（Characteristic Function）

    - 定义：$\phi_X(v) = E[e^{jvX}]$（$j$ 为虚数单位）。
    - 性质：
        - 一一对应：特征函数与分布函数（PDF）是一对傅里叶变换对。
        - 独立和的性质：若 $X, Y$ 独立，则 $\phi_{X+Y}(v) = \phi_X(v)\phi_Y(v)$。
        - 矩生成特性：$E(X^k) = (-j)^k \phi_X^{(k)}(0)$。

2. 母函数（Generating Function）

    - 主要用于非负整数随机变量。
    - 定义：$P(s) = E[s^X] = \sum p_k s^k$。
    - 性质：$E(X) = P'(1)$，$D(X) = P''(1) + P'(1) - [P'(1)]^2$。
    - 随机和的母函数：若 $Y = \sum_{i=1}^N X_i$（$N$ 也是随机变量），则 $G_Y(s) = G_N(G_X(s))$。

## 条件数学期望

这是现代概率论和随机过程的核心概念。

1. 定义

    - $E(X|Y=y)$ 是一个关于 $y$ 的函数。
    - $E(X|Y)$ 是一个随机变量，它是 $Y$ 的函数。

2. 全期望公式（Total Expectation）

    - $E[E(X|Y)] = E(X)$。即条件期望的期望等于无条件期望。这一性质在计算复杂期望时非常有用。

3. 性质

    - $E(g(Y)X | Y) = g(Y)E(X|Y)$（已知条件可以提出来）。
    - 若 $X, Y$ 独立，则 $E(X|Y) = E(X)$。
