根据提供的 PPT 资料（尤其是第一次课--概率论基础PPT.pdf 和刘次华随机过程C1.pdf），典型的随机变量分布分为离散型和连续型两大类。以下是这些典型分布的定义及其数学期望（Expectation）和方差（Variance）的详细总结：

## 离散型随机变量（Discrete Random Variables）

离散型随机变量的分布通常由分布律 $P{X=k}$ 描述。

### 1. 0-1 分布（两点分布 / Bernoulli Distribution）

- 定义：随机变量 $X$ 只取 0 和 1 两个值。$P(X=1)=p, P(X=0)=q$，其中 $0<p<1, p+q=1$。
- 数学期望 $E(X)$：$p$
- 方差 $D(X)$：$pq$（或 $p(1-p)$）
- 来源引用：，

### 2. 二项分布（Binomial Distribution）

- 符号：$B(n, p)$
- 定义：$n$ 重伯努利试验中成功的次数。分布律为 $P{X=k} = C_n^k p^k q^{n-k}$（$k=0, 1, \dots, n$）。
- 数学期望 $E(X)$：$np$
- 方差 $D(X)$：$npq$
- 来源引用：，

### 3. 泊松分布（Poisson Distribution）

- 符号：$P(\lambda)$
- 定义：常用于描述某段时间内源源不断到达的质点流个数。分布律为 $P{X=k} = \frac{\lambda^k}{k!}e^{-\lambda}$（$k=0, 1, 2, \dots, \lambda > 0$）。
- 数学期望 $E(X)$：$\lambda$
- 方差 $D(X)$：$\lambda$
- 来源引用：，

### 4. 几何分布（Geometric Distribution）

- 符号：$G(p)$
- 定义：在独立重复试验中，首次成功所需的试验次数。分布律为 $P{X=k} = q^{k-1}p$（$k=1, 2, \dots$）。具有无记忆性。
- 数学期望 $E(X)$：$\frac{1}{p}$
- 方差 $D(X)$：$\frac{q}{p^2}$
- 来源引用：，

### 5. 超几何分布（Hypergeometric Distribution）

- 符号：$H(n, N, M)$
- 定义：$N$ 件产品中有 $M$ 件次品，不放回地抽取 $n$ 件，其中含有的次品数 $X$。分布律为 $P{X=k} = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n}$。
- 备注：资料中给出了定义，但未在汇总表中列出其期望和方差公式。通常情况下 $E(X) = n\frac{M}{N}$。
- 来源引用：

## 连续型随机变量（Continuous Random Variables）

连续型随机变量的分布由概率密度函数 $f(x)$ 描述。

### 1. 均匀分布（Uniform Distribution）

- 符号：$U(a, b)$
- 定义：在区间 $[a, b]$ 上概率密度为常数。$f(x) = \frac{1}{b-a}$（$a \le x \le b$），其他为 0。
- 数学期望 $E(X)$：$\frac{a+b}{2}$
- 方差 $D(X)$：$\frac{(b-a)^2}{12}$
- 来源引用：，

### 2. 指数分布（Exponential Distribution）

- 符号：$E(\lambda)$
- 定义：常用于描述寿命、等待时间等。概率密度为 $f(x) = \lambda e^{-\lambda x}$（$x > 0, \lambda > 0$）。具有无记忆性。
- 数学期望 $E(X)$：$\frac{1}{\lambda}$
- 方差 $D(X)$：$\frac{1}{\lambda^2}$
- 来源引用：，

### 3. 正态分布（Normal Distribution）

- 符号：$N(\mu, \sigma^2)$
- 定义：概率密度为 $f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$。
- 数学期望 $E(X)$：$\mu$
- 方差 $D(X)$：$\sigma^2$
- 来源引用：，

## 总结表（基于资料）

| 分布类型 | 名称 | 符号 | 期望 $E(X)$ | 方差 $D(X)$ |
|:--|:--|:--|:--|:--|
| 离散型 | 0-1 分布 | - | $p$ | $pq$ |
| | 二项分布 | $B(n,p)$ | $np$ | $npq$ |
| | 泊松分布 | $P(\lambda)$ | $\lambda$ | $\lambda$ |
| | 几何分布 | $G(p)$ | $1/p$ | $q/p^2$ |
| 连续型 | 均匀分布 | $U(a,b)$ | $(a+b)/2$ | $(b-a)^2/12$ |
| | 指数分布 | $E(\lambda)$ | $1/\lambda$ | $1/\lambda^2$ |
| | 正态分布 | $N(\mu, \sigma^2)$ | $\mu$ | $\sigma^2$ |

（注：表中 $q = 1-p$）
