根据《2025 复习概述》和课件 `mac02.pdf`，0.618 法（黄金分割法）是第 2 章"单变量求极值"中的核心考点，属于精确一维搜索算法。

该方法的核心在于利用黄金分割比 $\frac{\sqrt{5}-1}{2} \approx 0.618$ 来选取试探点，每次迭代只需计算一次新的函数值即可缩小搜索区间。

以下是详细的解题步骤：

### 1. 准备工作

- 适用范围：目标函数 $\phi(\alpha)$ 在区间 $[a, b]$ 上是单峰函数。
- 关键常数：记 $t \approx 0.618$，则 $1-t \approx 0.382$。

### 2. 算法步骤（Algorithm 2.2）

请重点背诵以下流程，尤其是区间更新和试探点复用的逻辑：

#### 第 0 步：初始化

- 确定初始区间 $[a_0, b_0]$ 和终止精度 $\epsilon > 0$。
- 计算两个初始试探点 $p_0$ 和 $q_0$ 及其函数值：
    - 左试探点：$p_0 = a_0 + 0.382 \times (b_0 - a_0)$
    - 右试探点：$q_0 = a_0 + 0.618 \times (b_0 - a_0)$
    - 计算 $\phi(p_0)$ 和 $\phi(q_0)$。
- 置迭代计数 $k=0$。

#### 第 1 步：比较与判断

比较两个试探点的函数值 $\phi(p_k)$ 和 $\phi(q_k)$，决定缩短区间的方向：

- 情形 A：若 $\phi(p_k) \le \phi(q_k)$（左低右高）

    - 说明极小值在左侧，去掉右边的区间 $(q_k, b_k]$。
    - 新区间：$[a_{k+1}, b_{k+1}] = [a_k, q_k]$。
    - 终止检验：若区间长度 $b_{k+1} - a_{k+1} \le \epsilon$，停止迭代，取 $p_k$ 作为近似解。
    - 点复用与更新：
        - 旧的左点变新的右点：$q_{k+1} = p_k$，且 $\phi(q_{k+1}) = \phi(p_k)$（无需重算）。
        - 计算新的左点：$p_{k+1} = a_{k+1} + 0.382 \times (b_{k+1} - a_{k+1})$。
        - 计算 $\phi(p_{k+1})$。

- 情形 B：若 $\phi(p_k) > \phi(q_k)$（左高右低）

    - 说明极小值在右侧，去掉左边的区间 $[a_k, p_k)$。
    - 新区间：$[a_{k+1}, b_{k+1}] = [p_k, b_k]$。
    - 终止检验：若区间长度 $b_{k+1} - a_{k+1} \le \epsilon$，停止迭代，取 $q_k$ 作为近似解。
    - 点复用与更新：
        - 旧的右点变新的左点：$p_{k+1} = q_k$，且 $\phi(p_{k+1}) = \phi(q_k)$（无需重算）。
        - 计算新的右点：$q_{k+1} = a_{k+1} + 0.618 \times (b_{k+1} - a_{k+1})$。
        - 计算 $\phi(q_{k+1})$。

#### 第 2 步：循环

- 令 $k = k+1$，转回第 1 步继续比较。

### 3. 解题技巧与记忆口诀

- "去高留低"：函数值哪边大，就去掉哪一边的区间（因为是求极小值）。
- "新旧互换"：
    - 去掉了右边，原来的"左点"变成新区间里的"右点"。
    - 去掉了左边，原来的"右点"变成新区间里的"左点"。
- 计算量：除了第 0 步需要算 2 次函数值外，以后每一步只算 1 次新的函数值。

### 4. 考试计算题提示

如果考试要求手算，通常只要求迭代 1 到 2 步。例题逻辑（参考课件）：

1. 给定 $f(x) = x^2 - \sin(x)$，区间 $[0, 1]$。
2. 算出 $p_0 = 0.382, q_0 = 0.618$。
3. 算 $f(0.382)$ 和 $f(0.618)$。
4. 比较大小，确定新区间，计算新点。
