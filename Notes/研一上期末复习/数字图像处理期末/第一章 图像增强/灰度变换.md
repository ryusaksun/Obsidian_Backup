## 定义与原理

灰度变换是一种空域增强方法,它直接作用于图像的像素以改变其特性。

- 核心机制: 利用映射函数,将原始图像中每个像素的灰度值都映射到一个新的灰度值。
- 数学表达: 通常表示为 $t=T(s)$,其中 $s$ 是输入图像的灰度级,而 $t$ 是变换后的输出灰度级。

## 目的

- 强调细节: 将图像中用户关心的部分强调出来。
- 改善视觉效果: 对图像进行加工,以得到对具体应用来说视觉效果更好、更有用的图像。

## 常见的灰度变换函数

### 图像反转 (Image Negatives)

- 公式: $s=L-1-r$
- 作用: 用于反转图像的灰度级(如将黑变白,白变黑)。

### 对数变换 (Log Transformations)

- 公式: $s=c \log(1+r)$
- 作用: 扩展图像中暗像素的值,同时压缩高灰度级的值。

### 幂率(伽马)变换 (Power-Law Transformations)

- 公式: $s=cr^{\gamma}$
- 作用: 根据 $\gamma$ 值的不同(大于 1 或小于 1),可以分别用于抑制亮区或抑制暗区(即增亮暗区)。

### 分段线性变换 (Piecewise-Linear Transformation)

- 方法: 通过设定不同的线性函数段(如对比度拉伸、灰度切割等)来调节图像特定灰度范围的对比度。

## 基本概念

### 灰度 (Gray / Intensity)

灰度指的是图像中像素点的亮度或明暗程度。

- 在黑白(单色)图像中,灰度是描述图像信息的唯一属性。
- 涉及人眼对明暗(灰度)变化的感知能力。
- 在直方图中,灰度反映了图像中每种亮度出现的频率。

### 灰度级 (Gray Level)

灰度级是将灰度(亮度)进行离散化后的数值等级。

- 它是用来量化亮度的整数值。
- 范围: 从 0 到 $L-1$。
- 通常情况下(如 8 位图像),$L=256$,所以灰度级的范围是 0-255。
- 0 通常代表纯黑,$L-1$(如 255) 代表纯白。
- 直方图定义: 图像的灰度直方图以灰度级为横坐标($f=0, 1, ..., L-1$),以该灰度级出现的频率(像素个数)为纵坐标。

### 映射函数调整机制

灰度变换通过映射函数实现:

- 输入: 原始图像的灰度级(记为 $r$ 或 $s$)。
- 处理: 通过一个数学函数 $t = T(s)$ 或 $s = T(r)$ 进行计算。
- 输出: 得到新的灰度级(记为 $t$ 或 $s$),从而改变图像的视觉效果。

映射函数例子:

- 反转变换: $s = L-1-r$(用线性函数映射,将亮变暗,暗变亮)
- 对数变换: $s = c \log(1+r)$(用非线性函数映射,扩展低灰度值,压缩高灰度值)
- 幂率(伽马)变换: $s = c r^{\gamma}$(通过指数函数映射,用于校正亮度)

灰度变换就是设计一个映射函数,把像素原来的数值(Input Gray Level)换算成一个新的数值(Output Gray Level),以达到增强图像对比度或突出细节的目的。
