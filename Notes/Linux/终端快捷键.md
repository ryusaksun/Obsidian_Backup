## 光标导航与文本编辑

通过快捷键快速移动光标或修改命令，可以避免频繁使用方向键，下表总结了最常用的编辑指令：

|功能|快捷键|
|---|---|
|移动光标至行首|Ctrl + A csdn+1​|
|移动光标至行尾|Ctrl + E csdn+1​|
|向前/后跳过一个单词|Alt + F / Alt + B reddit+1​|
|删除光标前所有字符|Ctrl + U csdn+1​|
|删除光标后所有字符|Ctrl + K csdn+1​|
|粘贴上次删除的内容|Ctrl + Y [csdn](https://blog.csdn.net/qq_26399665/article/details/81063211)​|
|自动补全命令或路径|Tab [gist.github](https://gist.github.com/zhulianhua/befb8f61db8c72b4763d)​|
|引用上条命令的参数|Alt + . [csdn](https://blog.csdn.net/qq_26399665/article/details/81063211)​|

## 进程控制与屏幕管理

在运行程序或管理输出内容时，以下快捷键是控制终端状态的核心：

- **终止与挂起**：`Ctrl + C` 用于强制终止当前运行的进程；`Ctrl + Z` 将进程挂起至后台 。copilot.tencent+1​
    
- **退出与清理**：`Ctrl + D` 表示发送 EOF（文件结束符），通常用于退出当前会话或关闭标签页；`Ctrl + L` 清除当前屏幕内容，等同于执行 `clear` 命令 。bilibili+1​
    
- **流控制**：`Ctrl + S` 可阻止屏幕输出（常用于长时间输出时的暂停），按 `Ctrl + Q` 恢复输出 。csdn+1​
    

## 窗口与标签页管理

不同操作系统的终端在窗口操作上略有差异，但通常遵循以下逻辑：

- **Linux/Windows Terminal**：使用 `Ctrl + Shift + T` 新建标签页，`Ctrl + Shift + W` 关闭标签页 。在 Windows Terminal 中，复制和粘贴通常使用 `Ctrl + Shift + C` 和 `Ctrl + Shift + V` 。csdn+2​
    
- **macOS Terminal**：主要使用 Command (⌘) 键，如 `Cmd + T` 新建标签页，`Cmd + N` 新建窗口，`Cmd + W` 关闭当前视图 。[ftp-mac](https://ftp-mac.com/zh/mac-terminal-commands.html)​
    
- **标签切换**：在多个标签页间跳转通常使用 `Ctrl + PageUp / PageDown` 或 `Alt + 数字键` 。csdn+1​

## 补充
Warp 终端（Warp Terminal）以其现代化的交互体验而闻名，提供了许多快捷键来提高效率。

常用的快捷键可以分为**基础操作**、**分屏与标签页**、**光标与编辑**、以及 **AI 辅助**几大类：

## 1. 核心与 AI 功能

Warp 最具特色的功能通常通过以下快捷键触发：

- **AI 命令搜索 (AI Command Search)**: `Ctrl` + `~` (或输入 `#`)
    
    - 直接用自然语言描述你想做的事（例如 "撤销上一次 git 提交"），AI 会生成对应的命令 。sspai+1​
        
- **命令面板 (Command Palette)**: `Cmd` + `P` (macOS) / `Ctrl` + `Shift` + `P` (Linux/Windows)
    
    - 这是 Warp 的万能入口，可以搜索并执行 Warp 的所有功能和设置 。warp+1​
        
- **工作流 (Workflows)**: `Ctrl` + `Shift` + `R`
    
    - 打开侧边栏，浏览常用命令清单（如 git, docker, npm 的常见操作）。warp+1​
        

## 2. 分屏与标签页管理 (Panes & Tabs)

像浏览器一样管理你的终端窗口：

- **新建标签页**: `Cmd` + `T`[errorbaker](https://blog.errorbaker.tw/posts/benben/12-warp/)​
    
- **垂直分屏** (左右): `Cmd` + `D`errorbaker+1​
    
- **水平分屏** (上下): `Cmd` + `Shift` + `D`[errorbaker](https://blog.errorbaker.tw/posts/benben/12-warp/)​
    
- **关闭当前分屏/标签**: `Cmd` + `W`[errorbaker](https://blog.errorbaker.tw/posts/benben/12-warp/)​
    
- **切换分屏**: `Cmd` + `Opt` + `方向键` (上下左右)[errorbaker](https://blog.errorbaker.tw/posts/benben/12-warp/)​
    
- **切换标签页**: `Cmd` + `Shift` + `[` (左) / `]` (右)[errorbaker](https://blog.errorbaker.tw/posts/benben/12-warp/)​
    

## 3. Block（块）操作

Warp 将每次命令输入和输出视为一个独立的 "Block"，你可以像操作文本编辑器一样操作它们：

- **复制当前 Block 内容**: `Cmd` + `C` (选中后)
    
- **复制命令输出**: `Cmd` + `Shift` + `C` (或者通过菜单选择 "Copy Command Output")[csdn](https://blog.csdn.net/weixin_41905135/article/details/147782719)​
    
- **清除屏幕 (保留当前输入)**: `Cmd` + `K`[csdn](https://blog.csdn.net/weixin_41905135/article/details/147782719)​
    

## 4. 文本编辑与光标移动

Warp 的输入框支持类似 IDE 的编辑体验，支持鼠标点击定位，也支持标准快捷键：

- **移动到行首/行尾**: `Cmd` + `Left` / `Cmd` + `Right`
    
- **按单词移动**: `Opt` + `Left` / `Opt` + `Right`
    
- **多光标编辑**: 按住 `Opt` + 点击鼠标左键，可以在多个位置同时输入。
    

## 提示

如果你记不住所有快捷键，最重要的一条是 **`Cmd` + `P`**。打开命令面板后，你可以输入关键词（如 "split"）来查找对应的功能和快捷键提示 。javabetter+1​
