## 泊松过程的基本定义与性质

泊松过程 ${N(t), t \ge 0}$ 是一种特殊的计数过程，用于描述随机事件在时间轴上的发生次数。

- 定义条件：
    1. 初值：$N(0) = 0$。
    2. 独立增量：在不相交的时间区间内，事件发生的次数是相互独立的。
    3. 平稳增量：在时间间隔 $(t, t+s)$ 内事件发生的次数仅与 $s$ 有关，与 $t$ 无关。
    4. 小概率性质：在极短时间 $\Delta t$ 内：
        - 出现 1 次事件的概率为 $\lambda \Delta t + o(\Delta t)$。
        - 出现 2 次及以上事件的概率为 $o(\Delta t)$。

## 概率分布与数字特征

这是计算题的基础，必须熟记公式。

- 概率分布律：时刻 $t$ 发生的事件数 $N(t)$ 服从参数为 $\lambda t$ 的泊松分布：

    $$P{N(t) = n} = \frac{(\lambda t)^n}{n!} e^{-\lambda t}, \quad n=0, 1, 2, \dots$$

    其推导基于递推微分方程：$P_0'(t) = -\lambda P_0(t)$ 和 $P_n'(t) = -\lambda P_n(t) + \lambda P_{n-1}(t)$。

- 数字特征：

    - 均值：$E[N(t)] = \lambda t$
    - 方差：$D[N(t)] = \lambda t$
    - 自相关函数：$R(t_1, t_2) = \lambda^2 t_1 t_2 + \lambda \min(t_1, t_2)$。

## 到达时间与间隔分布（重要考点）

这部分涉及事件发生的时间特性。

- 第 1 个事件到达时间 $T_1$：服从参数为 $\lambda$ 的负指数分布。概率密度函数为 $f(t) = \lambda e^{-\lambda t}$。

- 第 $n$ 个事件到达时间 $T_n$：服从参数为 $n, \lambda$ 的爱尔朗分布（Erlang Distribution）（或 Gamma 分布）。概率密度为：

    $$f(t) = \frac{\lambda (\lambda t)^{n-1}}{(n-1)!} e^{-\lambda t}$$

- 条件分布性质（均匀分布特性）：如果已知在 $(0, t)$ 内发生了 1 个事件，则该事件发生的时间 $S$ 在 $(0, t)$ 上服从均匀分布。即 $f_{S|N(t)=1}(s) = \frac{1}{t}, 0 < s < t$。推广：已知 $(0, t)$ 内发生 $n$ 次事件，则这 $n$ 次事件的发生时间可视作 $n$ 个独立同分布于 $(0, t)$ 的均匀随机变量的顺序统计量。

## 非齐次泊松过程（Non-homogeneous Poisson Process）

当强度 $\lambda$ 不是常数，而是随时间变化的函数 $\lambda(t)$ 时：

- 均值函数：$m(t) = \int_0^t \lambda(s) ds$。
- 概率分布：

    $$P{N(t) = n} = \frac{[m(t)]^n}{n!} e^{-m(t)}$$

- 应用举例：商店在不同时段（如早高峰、午休）顾客到达率不同，需用积分求出某时段内的平均到达人数 $m(t)$ 来计算概率。

## 泊松过程的变换（和与差）

- 和：两个独立的泊松过程（参数 $\lambda_1, \lambda_2$）之和 $N_1(t) + N_2(t)$ 仍然是泊松过程，参数为 $\lambda_1 + \lambda_2$。

- 差：两个独立泊松过程之差 $Y(t) = N_1(t) - N_2(t)$ 不是泊松过程。
    - 均值：$E[Y(t)] = (\lambda_1 - \lambda_2)t$
    - 方差：$Var[Y(t)] = (\lambda_1 + \lambda_2)t$。

## 复合泊松过程（Compound Poisson Process）

这是复习大纲重点强调的内容（第 7-8 小节）。

- 定义：设有泊松过程 ${N(t), t \ge 0}$ 和一族独立同分布的随机变量 ${Y_n}$（如每次索赔的金额、每户移民的人数），且两者相互独立。则随机过程

    $$X(t) = \sum_{n=1}^{N(t)} Y_n$$

    称为复合泊松过程。若 $N(t)=0$，则 $X(t)=0$。

- 重要数字特征（必须掌握）：利用全期望公式和全方差公式推导：

    - 均值：$E[X(t)] = \lambda t E[Y_1]$（解释：平均总发生值 = 平均发生次数 $\times$ 每次的平均值）
    - 方差：$D[X(t)] = \lambda t E[Y_1^2]$（注意：这里是 $Y$ 的二阶原点矩 $E[Y^2]$，而不是方差 $D[Y]$）。

- 应用举例（Source 例 2）：
    - 场景：移民户数是 $\lambda=2$ 的泊松过程，每户人数 $Y_n$ 服从特定分布（如 1人-p, 2人-q 等）。
    - 问题：求 $t$ 时间内移民总人数 $X(t)$ 的均值和方差。
    - 解法：先算每户人数的 $E[Y]$和 $E[Y^2]$，然后直接代入上述公式 $E[X(t)] = \lambda t E[Y]$ 和 $D[X(t)] = \lambda t E[Y^2]$。

## 总结与复习建议

根据大纲要求，您需要：

1. 背诵公式：泊松分布概率公式、$E[N(t)]$、$D[N(t)]$、指数分布密度函数。
2. 掌握复合泊松过程计算：特别是均值和方差的公式（$E[X]=\lambda t E[Y]$, $D[X]=\lambda t E[Y^2]$）。
3. 看懂例题：重点看课件中的例 1（非齐次泊松过程计算概率）和例 2（复合泊松过程计算均值方差）。
