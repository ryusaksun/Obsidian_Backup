## 一、DFT(离散傅里叶变换)、DCT(离散余弦变换)的原理和公式

根据《数字图像处理:第三章傅里叶变换.pdf》的内容,以下是DFT(离散傅里叶变换)和DCT(离散余弦变换)的核心原理与公式总结。这两者是图像频域处理和压缩的基础。

### DFT (离散傅里叶变换)

**原理:**

- **物理意义**: DFT将图像从空间域(Spatial Domain)转换到频率域(Frequency Domain)。
    - 图像中灰度变化缓慢的区域对应低频信息。
    - 灰度变化剧烈的区域(如边缘、噪点)对应高频信息。
- **基函数**: 任何函数波形可以表示为一系列正弦和余弦波的叠加。傅里叶变换就是求这些波的系数。
- **复数形式**: DFT的结果通常是复数,包含实部 $R(u,v)$ 和虚部 $I(u,v)$。
    - **幅度谱**(Spectrum): $|F(u,v)| = [R(u,v)^2 + I(u,v)^2]^{1/2}$,表示各频率分量的强弱。
    - **相位谱**(Phase): $\phi(u,v) = \arctan[I(u,v)/R(u,v)]$,决定了波的位置。

**公式:**

讲义中给出了幺正变换(Unitary Transform)的形式(即正反变换系数对称,均为 $1/\sqrt{MN}$),但同时也指出系数乘积满足 $1/MN$ 即可

- **一维 DFT 定义**:
    $$F(u) = \frac{1}{N} \sum_{k=0}^{N-1} f(k) e^{-j \frac{2\pi uk}{N}}, \quad u=0,1,\dots,N-1$$
    *(注: PPT第19页一维公式系数为1/N,第20页二维公式系数为 $1/\sqrt{MN}$,考试时请以此为准或遵循老师上课强调的习惯)*
- **二维 DFT 定义**(常用):
    $$F(u,v) = \frac{1}{\sqrt{MN}} \sum_{x=0}^{N-1} \sum_{y=0}^{M-1} f(x,y) e^{-j 2\pi (\frac{ux}{N} + \frac{vy}{M})}$$
- **二维 IDFT (逆变换) 定义**:
    $$f(x,y) = \frac{1}{\sqrt{MN}} \sum_{u=0}^{N-1} \sum_{v=0}^{M-1} F(u,v) e^{j 2\pi (\frac{ux}{N} + \frac{vy}{M})}$$

### DCT (离散余弦变换)

**原理:**

- **解决的问题**: DFT的参数是复数,数据描述相当于实数的两倍。为了减少数据量,产生了实数域的DCT变换。
- **构造思想**: 由于实偶函数的傅里叶变换只含有余弦项(实数),DCT的基本思想是将一个实函数进行对称延拓变成一个实偶函数,从而利用傅里叶变换得到实偶结果。
- **能量集中特性**:
    - DFT变换后的数据集中在频谱图的中间(低频在中心)
    - DCT变换后的数据高度集中在频谱图的左上角(低频)。对于平滑图像,DCT能更有效地把信息封装在少数系数中,非常适合图像压缩(如JPEG标准)

**公式:**

DCT变换核为余弦函数

- **一维 DCT 定义**:
    $$F(u) = c(u) \sum_{x=0}^{N-1} f(x) \cos \left[ \frac{(2x+1)u\pi}{2N} \right]$$
    其中系数 $c(u)$:
    $$c(u) = \begin{cases} \sqrt{\frac{1}{N}}, & \text{当 } u=0 \\ \sqrt{\frac{2}{N}}, & \text{当 } u \neq 0 \end{cases}$$
- **二维 DCT 定义**:
    $$F(u,v) = c(u)c(v) \sum_{x=0}^{M-1} \sum_{y=0}^{N-1} f(x,y) \cos \left[ \frac{(2x+1)u\pi}{2M} \right] \cos \left[ \frac{(2y+1)v\pi}{2N} \right]$$
    *(注: PPT中公式截图较为模糊,但标准公式结构即为两个余弦函数的乘积)*

### 总结对比

| 特性 | DFT (离散傅里叶变换) | DCT (离散余弦变换) |
|:--|:--|:--|
| **数值域** | 复数 (实部+虚部) | 实数 |
| **能量分布** | 集中在中心 (需频谱移位) | 集中在左上角 |
| **应用场景** | 图像滤波、分析、卷积 | 图像压缩 (JPEG) |
| **基函数** | 正弦 + 余弦 ($e^{-jx}$) | 仅余弦 ($\cos$) |

## 二、正交变换后频谱图像的特征，卷积定理等

根据《数字图像处理:第三章傅里叶变换.pdf》等资料,以下是关于正交变换后频谱图像特征及卷积定理的复习要点:

### 正交变换后频谱图像的特征

通常指的是离散傅里叶变换 (DFT) 后的频谱表现

- **频率分布特征**:
    - **低频分量(中心)**: 对应图像中灰度变化缓慢的区域(如平坦的背景)。在经过频谱移位(`fftshift`)后,直流分量(DC,频率为0)位于频谱图的中心,通常是图像中最亮的一个点,代表图像的平均灰度。
    - **高频分量(四周)**: 对应图像中灰度变化剧烈的区域(如边缘、噪点、细节)。频谱图越靠外,频率越高。
- **物理意义**:
    - **幅度谱 (Magnitude/Spectrum)**: $|F(u,v)|$,表示各频率分量的强弱,决定了图像中某种纹理或灰度变化的强烈程度。我们在屏幕上看到的频谱图通常就是幅度谱。
    - **相位谱 (Phase)**: $\phi(u,v)$,包含图像的位置信息,决定了正弦波在图像中的具体位置。虽然肉眼看不出明显规律,但对恢复图像至关重要。
- **几何变换性质**:
    - **平移不变性**: 图像在空间域平移,其频谱的幅度谱不变,只有相位谱发生变化。这意味着无论物体在图像的哪个位置,其频谱图看起来是一样的。
    - **旋转性**: 图像在空间域旋转一定角度,其频谱在频域也会旋转相同的角度。
    - **可分离性**: 二维变换可以分解为两次一维变换(先对行做变换,再对列做变换),这大大提高了计算效率。

### 卷积定理 (Convolution Theorem)

这是频域滤波的理论基础,必须熟记公式和物理意义

- **定义**: 空间域的卷积等于频率域的乘积。
    - 公式表示: $f(x,y) * h(x,y) \Leftrightarrow F(u,v)H(u,v)$。
    - 反之亦然: 空间域的乘积等于频率域的卷积。
- **意义**:
    - 直接在空间域进行卷积运算(如使用大模板滤波)计算量非常大。
    - 利用卷积定理,我们可以将图像转换到频域,直接与滤波器的频率响应函数(如低通或高通滤波器)相乘,最后再反变换回空间域。这使得复杂的滤波操作变得简单且快速。

### DCT (离散余弦变换) 的频谱特征(对比记忆)

- **实数域变换**: 与DFT不同,DCT是实数变换,没有虚部,计算速度更快。
- **能量集中特性**: DCT变换后的数据高度集中在频谱图的左上角(低频区域)。
    - **DFT**: 能量集中在中心(移位后)。
    - **DCT**: 能量集中在左上角。
    - 这种特性使得DCT非常适合图像压缩(如JPEG),因为可以去除右下角(高频)的系数而对图像质量影响较小。

### 复习总结

- **简答题**可能会问: 频谱图中心亮斑代表什么?(答:低频/直流分量/平均灰度)或者平移图像后频谱图会变吗?(答:幅度谱不变,相位谱变)。
- **原理题**可能会让你解释卷积定理的作用(答:将空域复杂的卷积运算转化为频域简单的乘法运算,加速处理)。

## 三、频域的高通滤波和低通滤波，原理，区别与联系等

根据《数字图像处理:第三章傅里叶变换.pdf》及复习大纲,频域的高通与低通滤波是基于图像频率特征的处理方法。以下是针对考试的详细复习总结:

### 核心原理:什么是图像的频率?

在频域处理中,频率反映了图像灰度变化的快慢:

- **低频分量**: 对应图像中灰度变化缓慢的区域,如平坦的背景、大片色块。在频谱图中通常位于中心区域(经过移位后)。
- **高频分量**: 对应图像中灰度变化剧烈的区域,如边缘、细节、噪点。在频谱图中通常位于四周边缘区域。

**处理基础(卷积定理):**

频域滤波的基础是卷积定理:空域的卷积等于频域的乘积。

这意味着我们不需要在空域进行复杂的模板卷积,只需将图像变换到频域,乘以滤波器的转移函数 $H(u,v)$,再反变换回去即可。

### 低通滤波 (Low-Pass Filtering)

- **原理**: 让频谱图中心部分的低频分量通过,衰减或抑制四周的高频分量。
- **物理意义**: 去除了图像中变化剧烈的部分(如边缘和噪声)。
- **视觉效果**: 图像变模糊,边缘变平滑。起到平滑图像和去除噪声的作用。
- **对应空域操作**: 相当于空域中的均值滤波或高斯平滑滤波(平滑滤波器)。
- **典型滤波器**: 理想低通、巴特沃斯低通、高斯低通。

### 高通滤波 (High-Pass Filtering)

- **原理**: 让频谱图四周的高频分量通过,衰减或抑制中心的低频分量(通常会把直流分量置0或很小)。
- **物理意义**: 去除了图像中平坦的背景,保留了变化剧烈的部分。
- **视觉效果**: 图像的边缘和细节被突出,背景变暗。起到锐化图像和提取边缘的作用。
- **对应空域操作**: 相当于空域中的梯度算子(如Sobel)或拉普拉斯算子(锐化滤波器)。
- **典型滤波器**: 理想高通、巴特沃斯高通、高斯高通。

### 区别与联系(★考试对比总结表)

| 特性 | **低通滤波 (Low-Pass)** | **高通滤波 (High-Pass)** |
|:--|:--|:--|
| **保留频率** | 低频(中心区域) | 高频(四周区域) |
| **抑制频率** | 高频(四周区域) | 低频(中心区域) |
| **视觉效果** | **模糊**、平滑、去噪 | **锐化**、边缘增强、轮廓提取 |
| **对应空域** | 平滑滤波 (均值/中值) | 锐化滤波 (Sobel/Laplacian) |
| **应用场景** | 去除椒盐噪声、高斯噪声,磨皮 | 边缘检测,增强图像细节 |
| **直方图影响** | 可能会缩小灰度范围(对比度降低) | 直方图主要分布在低灰度区(背景变黑) |

### 复习建议(结合图片考点)

- **看图理解**: 在《第三章傅里叶变换.pdf》的第30页,有非常直观的对比图
    - **低通**: 那只青蛙的图像变得模糊,像是失焦了
    - **高通**: 青蛙的内部颜色没了,只剩下明亮的轮廓线条
- **记忆口诀**:
    - **低频** = **慢**变化 = **背景** = **平滑**
    - **高频** = **快**变化 = **边缘** = **锐化**
