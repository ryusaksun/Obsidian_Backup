这是一道使用内点法（通常指障碍函数法，Barrier Function Method）求解约束优化问题的计算题。

内点法的核心思想是构造一个包含障碍项（Barrier Term）的目标函数，将约束问题转化为无约束问题求解。随着障碍参数的变化，无约束问题的最优解会逼近原问题的最优解。

以下是详细的求解步骤：

## 第一步：构造障碍函数

对于不等式约束优化问题：

$$\min f(x) \quad \text{s.t.} \quad g(x) \ge 0$$

我们通常构造如下形式的障碍函数（这里以对数障碍函数为例，倒数障碍函数结果也是一样的）：

$$I(x, r) = f(x) - r \ln(g(x))$$

其中 $r > 0$ 是障碍参数。

代入题目中的函数：

$$f(x) = 2x_1 + 3x_2$$

$$g(x) = 1 - 2x_1^2 - x_2^2$$

构造函数：

$$I(x_1, x_2, r) = 2x_1 + 3x_2 - r \ln(1 - 2x_1^2 - x_2^2)$$

## 第二步：求解无约束极值

对 $I(x, r)$ 关于 $x_1$ 和 $x_2$ 分别求偏导数，并令其为 0：

1. 对 $x_1$ 求导：

    $$\frac{\partial I}{\partial x_1} = 2 - r \cdot \frac{1}{1 - 2x_1^2 - x_2^2} \cdot (-4x_1) = 0$$

    整理得：

    $$2 + \frac{4r x_1}{g(x)} = 0 \implies \frac{2r x_1}{g(x)} = -1 \quad \text{......①}$$

2. 对 $x_2$ 求导：

    $$\frac{\partial I}{\partial x_2} = 3 - r \cdot \frac{1}{1 - 2x_1^2 - x_2^2} \cdot (-2x_2) = 0$$

    整理得：

    $$3 + \frac{2r x_2}{g(x)} = 0 \implies \frac{2r x_2}{g(x)} = -3 \quad \text{......②}$$


## 第三步：寻找 $x_1$ 和 $x_2$ 的关系

观察方程 ① 和 ②，我们可以用 ① 除以 ② 来消去复杂的 $r$ 和 $g(x)$ 项：

$$\frac{\frac{2r x_1}{g(x)}}{\frac{2r x_2}{g(x)}} = \frac{-1}{-3}$$

$$\frac{x_1}{x_2} = \frac{1}{3} \implies x_2 = 3x_1$$

这一步告诉我们，无论 $r$ 取什么值，最优解一定在直线 $x_2 = 3x_1$ 上。

## 第四步：求极限（逼近边界）

内点法的原理是，当障碍参数 $r \to 0$ 时，无约束极值点会收敛到原问题的最优解。对于线性目标函数，最优解通常位于可行域的边界上。

即满足约束等式：

$$1 - 2x_1^2 - x_2^2 = 0$$

将 $x_2 = 3x_1$ 代入边界方程：

$$1 - 2x_1^2 - (3x_1)^2 = 0$$

$$1 - 2x_1^2 - 9x_1^2 = 0$$

$$1 - 11x_1^2 = 0$$

$$x_1^2 = \frac{1}{11}$$

解得：

$$x_1 = \pm \frac{1}{\sqrt{11}}$$

## 第五步：确定符号

回到原目标函数 $\min f(x) = 2x_1 + 3x_2$。

为了使目标函数最小，且 $x_1, x_2$ 同号（因为 $x_2 = 3x_1$），我们需要选取负值。

所以：

$$x_1 = -\frac{1}{\sqrt{11}}$$

$$x_2 = 3x_1 = -\frac{3}{\sqrt{11}}$$

## 最终答案

填空处应填：

$( -\frac{1}{\sqrt{11}}, -\frac{3}{\sqrt{11}} )$

_(注：如果你需要分母有理化，也可以写成 $( -\frac{\sqrt{11}}{11}, -\frac{3\sqrt{11}}{11} )$ )_

这是一道利用内点法（障碍函数法）求解不等式约束优化问题的题目。

根据图片中的纠正一下部分，我们按照修正后的约束条件来求解。

**问题描述：**

- 目标函数：$\min f(x) = x_1 + x_2$

- 约束条件（修正后）：

    - $g_1(x) = -x_1^2 + x_2 \ge 0$

    - $g_2(x) = -x_1 \ge 0$ （即 $x_1 \le 0$）


---

## 求解步骤

### 第一步：构造障碍函数

使用对数障碍函数，将约束条件放入目标函数中：

$$I(x, r) = f(x) - r \sum \ln(g_i(x))$$

$$I(x_1, x_2, r) = x_1 + x_2 - r \ln(-x_1^2 + x_2) - r \ln(-x_1)$$

其中 $r > 0$ 是障碍参数。

### 第二步：求解无约束极值

对 $I(x, r)$ 关于 $x_1$ 和 $x_2$ 求偏导数，并令其为 0。

1. 对 $x_2$ 求导：

    $$\frac{\partial I}{\partial x_2} = 1 - r \cdot \frac{1}{-x_1^2 + x_2} \cdot 1 = 0$$

    由此得到：

    $$\frac{r}{-x_1^2 + x_2} = 1 \implies -x_1^2 + x_2 = r \quad \text{......①}$$

2. 对 $x_1$ 求导：

    $$\frac{\partial I}{\partial x_1} = 1 - r \cdot \frac{1}{-x_1^2 + x_2} \cdot (-2x_1) - r \cdot \frac{1}{-x_1} \cdot (-1) = 0$$

    整理得：

    $$1 + \frac{2r x_1}{-x_1^2 + x_2} + \frac{r}{-x_1} = 0$$

    将方程 ① 中的结论 $\frac{r}{-x_1^2 + x_2} = 1$ 代入上式：

    $$1 + 2x_1 \cdot (1) - \frac{r}{x_1} = 0$$

    $$1 + 2x_1 - \frac{r}{x_1} = 0$$


### 第三步：解方程求 $x_1$

两边同乘 $x_1$（注意我们在内点，所以 $x_1 < 0$）：

$$x_1 + 2x_1^2 - r = 0$$

即：

$$2x_1^2 + x_1 - r = 0$$

解这个关于 $x_1$ 的一元二次方程：

$$x_1 = \frac{-1 \pm \sqrt{1 - 4(2)(-r)}}{4} = \frac{-1 \pm \sqrt{1 + 8r}}{4}$$

由于 $r > 0$，则 $\sqrt{1+8r} > 1$。

- 若取+号：$x_1 = \frac{-1 + (>1)}{4} > 0$，这不满足约束 $x_1 \le 0$。

- 若取-号：$x_1 = \frac{-1 - (>1)}{4} < 0$，满足约束。


所以最优路径上的点为：

$$x_1(r) = \frac{-1 - \sqrt{1 + 8r}}{4}$$

### 第四步：求极限得最优解

令障碍参数 $r \rightarrow 0$：

1. 求 $x_1^*$：

    $$x_1^* = \lim_{r \to 0} \frac{-1 - \sqrt{1 + 8r}}{4} = \frac{-1 - 1}{4} = -0.5$$

2. 求 $x_2^*$：

    由方程 ①：$x_2 = x_1^2 + r$

    代入极限值：

    $$x_2^* = (-0.5)^2 + 0 = 0.25$$

3. 求目标函数最小值：

    $$\min f(x) = x_1^* + x_2^* = -0.5 + 0.25 = -0.25$$



### 最终答案

填空处应填：

min f(x) = ( -0.25 ) ; [x1, x2] = [ (-0.5) , (0.25) ]
